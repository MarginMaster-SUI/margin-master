## 執行摘要

### 項目簡介

**MarginMaster** 是基於 Sui 區塊鏈和 DeepBook Margin 協議的**社交化保證金交易平台**，結合複製交易、AI 風險管理和交易者排行榜功能，旨在降低 DeFi 保證金交易門檻並提供透明的社交交易體驗。

### 一句話描述

> "Sui 生態的 eToro - 讓任何人都能通過跟隨頂級交易者參與 DeFi 保證金交易"

### 解決的問題

| 問題 | 當前狀況 | MarginMaster 解決方案 |
|------|---------|---------------------|
| **高門檻** | 保證金交易需要專業知識與經驗 | 一鍵跟單頂級交易者 |
| **高風險** | 新手容易因槓桿過度而爆倉 | AI 實時風險監控與警報 |
| **信息不對稱** | 難以找到並驗證成功交易者 | 透明的鏈上績效排行榜 |
| **缺乏學習途徑** | 沒有安全的練習環境 | 模擬交易模式 |
| **孤立交易** | 交易者缺乏變現渠道 | 跟單費用收入 + 社交聲譽 |

### 核心價值主張

| 維度 | 價值 |
|------|------|
| **時機優勢** | DeepBook Margin 於 2025/1 推出，市場空缺明顯 |
| **技術優勢** | Sui 低 gas + 快速確認，完美適配高頻跟單 |
| **商業模式** | 雙重收入：DeepBook 推薦獎勵 + 跟單費用分成 |
| **競爭優勢** | Sui 生態首個社交化保證金交易平台 |
| **可行性** | 基於成熟 SDK，7 天可完成 MVP |

## 核心功能

### 1. 保證金交易面板

**功能描述**：完整的 DeepBook Margin 交易界面

**核心特性**：
- ✅ **訂單類型**
  - 市價單 (Market Order)
  - 限價單 (Limit Order)
  - 計劃訂單（止損/止盈）

- ✅ **槓桿管理**
  - 1x - 10x 槓桿選擇
  - 動態槓桿調整
  - 槓桿使用率可視化

- ✅ **風險監控**
  - 實時風險比率顯示
  - 預估清算價格
  - 保證金使用情況
  - 健康度評分 (0-100)

- ✅ **持倉管理**
  - 當前持倉列表
  - 未實現盈虧
  - 一鍵平倉
  - 批量操作

**用戶價值**：
- 降低交易門檻，界面直觀易用
- 實時風險提示，減少爆倉風險
- 完整功能支持，無需切換平台

### 2. 社交跟單系統

**功能描述**：自動複製交易者的所有操作

**核心特性**：
- ✅ **一鍵跟單**
  - 選擇交易者 → 設置參數 → 啟用
  - 自動同步所有交易操作
  - 實時執行狀態通知

- ✅ **靈活配置**
  - 跟單比例：1% - 100%
  - 最大單筆倉位限制
  - 費率：0% - 10%（市場決定）
  - 停止條件設置

- ✅ **智能風險控制**
  - 跟單前自動風險檢查
  - 風險比率 < 1.2 自動拒絕
  - 餘額不足時暫停
  - 緊急停止功能

- ✅ **透明執行**
  - 每筆跟單詳細記錄
  - 成功/失敗原因說明
  - 費用明細展示
  - 績效統計

**用戶價值**：
- **Followers**：零經驗參與保證金交易
- **Leaders**：獲得被動收入，建立個人品牌
- **平台**：交易量增長，手續費收入

### 3. 交易者排行榜

**功能描述**：多維度展示頂級交易者

**核心特性**：
- ✅ **多維度排名**
  - 30 天盈虧 (PnL)
  - 勝率 (Win Rate)
  - 夏普比率 (Sharpe Ratio)
  - 最大回撤 (Max Drawdown)
  - 跟隨者數量
  - 總交易次數

- ✅ **篩選功能**
  - 交易風格（激進/保守/平衡）
  - 最低勝率
  - 最低 Followers
  - 費率範圍
  - 驗證狀態

- ✅ **詳細檔案頁**
  - 績效折線圖（日/週/月）
  - 交易歷史記錄
  - 持倉分布圖
  - Followers 列表
  - 交易者說明
  - 風格標籤

**數據來源**：
- DeepBook 鏈上交易數據（通過 Surflux Indexer）
- MarginMaster 自建跟單數據
- 實時計算績效指標

**用戶價值**：
- 快速發現優質交易者
- 透明的歷史績效
- 降低信息不對稱

### 4. AI 風險管理

**功能描述**：智能風險評估與警報系統

**核心特性**：
- ✅ **實時風險評分**
  - 健康度：0-100 分
  - 風險等級：安全/警告/危險
  - 清算距離：%

- ✅ **智能警報系統**
  - 🔴 **危險警報**（健康度 < 30）
    - 接近清算價格
    - 立即推送通知
    - 建議平倉或增加保證金

  - 🟡 **警告提醒**（健康度 30-50）
    - 保證金使用率過高
    - 建議減倉或調整槓桿

  - 🟢 **信息提示**（健康度 > 50）
    - 市場波動率變化
    - 最佳開倉時機

- ✅ **自動化建議**
  - 建議止損價位（基於 ATR）
  - 建議止盈價位（基於 R/R 比）
  - 建議減倉比例
  - 對沖策略推薦

- ✅ **風險分析**
  - 組合波動率
  - 集中度風險
  - 槓桿使用率
  - 時間衰減風險

**算法基礎**：
```typescript
健康度 = f(風險比率, 槓桿倍數, 市場波動率, 持倉時間)

其中：
- 風險比率權重：50%
- 槓桿倍數權重：25%
- 市場波動率權重：15%
- 持倉時間權重：10%
```

**用戶價值**：
- 及時發現風險，避免爆倉
- 數據驅動的決策支持
- 提升整體交易勝率

### 5. 模擬交易模式

**功能描述**：零風險的練習環境

**核心特性**：
- ✅ **虛擬資金**
  - 初始 $10,000 虛擬 USDC
  - 可重置
  - 獨立於真實資金

- ✅ **真實市場數據**
  - 使用 DeepBook 實時價格
  - 真實的訂單簿深度
  - 實際的滑點模擬

- ✅ **完整功能**
  - 所有交易功能可用
  - 跟單系統可用（虛擬跟單）
  - 風險管理系統可用

- ✅ **績效追蹤**
  - 模擬交易歷史
  - 績效統計
  - 學習進度分析

- ✅ **平滑升級**
  - 一鍵切換到真實模式
  - 保留設置和偏好

**用戶價值**：
- 新手無風險學習
- 策略測試環境
- 降低平台進入門檻

## 市場分析

### 目標市場規模

**TAM (Total Addressable Market) - 全球市場**
```
全球 DeFi 用戶：~7,000,000
× 保證金交易參與率：15%
= 潛在用戶：~1,000,000
× 年均交易額：$50,000
= TAM：$50B
```

**SAM (Serviceable Addressable Market) - Sui 生態**
```
Sui 生態 DeFi 用戶：~500,000
× DeFi 活躍用戶比例：20%
= 可觸及用戶：~100,000
× 年均交易額：$30,000
= SAM：$3B
```

**SOM (Serviceable Obtainable Market) - MarginMaster 目標**
```
Year 1 目標市場份額：5%
= 首年目標用戶：~5,000
× 年均交易額：$20,000
= SOM：$100M
× 平台手續費率：0.5%
= 預期年收入：$500K
```

## 商業模式

### 收入來源

#### 1. 跟單費用分成（主要）

**模型**：
```
Follower 支付跟單費
    ↓
Protocol Fee (5%)
    ↓
Leader Fee (95%)

示例：
- 跟單交易量：$10,000
- Leader 費率：5%
- 總費用：$500
- Protocol 收入：$25
- Leader 收入：$475
```

**預估收入**：
```
Year 1:
- 活躍跟單關係：100
- 月均跟單交易量/關係：$50,000
- 平均費率：5%
- 月總費用：$250,000
- Protocol 收入 (5%)：$12,500/月
- 年收入：$150,000
```

#### 2. DeepBook 推薦獎勵

**模型**：
```
用戶在 MarginMaster 交易
    ↓
DeepBook 手續費
    ↓
推薦返佣 (10-20%)
    ↓
MarginMaster 收入
```

**預估收入**：
```
Year 1:
- 月交易量：$10M
- DeepBook 手續費：0.1%
- 總手續費：$10,000
- 推薦返佣 (15%)：$1,500/月
- 年收入：$18,000
```

#### 3. 策略市場佣金（Phase 2）

**模型**：
```
Leader 上架付費策略
    ↓
Follower 訂閱使用
    ↓
Platform Fee (20%)
```

**預估收入**：
```
Year 2:
- 上架策略數：50
- 月均訂閱收入/策略：$500
- 總訂閱收入：$25,000
- Platform Fee (20%)：$5,000/月
```
